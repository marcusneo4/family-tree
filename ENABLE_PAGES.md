## Step 1: Enable GitHub Pages

1. Go to your repository: https://github.com/marcusneo4/family-tree
2. Click on the **"Settings"** tab (at the top of the repository page)
3. Scroll down in the left sidebar and click **"Pages"**
4. Under **"Source"** section:
   - Select **"Deploy from a branch"**
   - Branch: Choose **`main`**
   - Folder: Choose **`/ (root)`**
5. Click **"Save"** button
6. Wait 1-2 minutes for GitHub to build your site

## Step 2: Access Your Website

After enabling Pages, GitHub will provide you with a URL like:
```
https://marcusneo4.github.io/family-tree/
```

Your website will be live at this URL!

**Note:** It may take 1-5 minutes after enabling for the site to be available. You'll see a green checkmark when it's ready.

## Step 3: Future Updates

Whenever you make changes to your website:

1. Make your changes locally
2. Commit and push:
   ```powershell
   git add .
   git commit -m "Your update message"
   git push
   ```
3. Your website will automatically update within 1-2 minutes!
